# hybrid-rag-hallucination-verification
The module is specifically tailored for the Russian language using the SberQuAD dataset.





Проект представляет собой продвинутую **RAG (Retrieval-Augmented Generation)** систему для ответов на вопросы на русском языке на базе датасета **SberQuAD**.

Особенность пайплайна — **гибридный поиск** (лексический + семантический) и **двухуровневая верификация ответов** (внутренний Критик с Chain-of-Thought и внешний Судья) для надежной блокировки галлюцинаций LLM.

---

## Основные фичи

* **Гибридный поиск (Hybrid Search):** Сочетает сильные стороны точного совпадения слов (**BM25** с лемматизацией через `pymorphy3`) и глубокого семантического понимания контекста (**LaBSE** эмбеддинги).
* **Локальная LLM:** Использование **Qwen2.5-7B-Instruct**- одной из лучших открытых моделей для работы с русским языком.
* **Модуль безопасности (Self-Verification Critic):** Встроенный агент-валидатор проверяет сгенерированный ответ по фактам из контекста и жестко блокирует выдачу в случае обнаружения бреда или галлюцинаций.
* **Оценка качества (Evaluation Pipeline):** Скрипт автоматического сбора логов в Excel (`.xlsx`) для ручной разметки человеком и последующего расчета метрик классификации (`Accuracy`, `Precision`, `Recall`, `F1-score`, `Confusion Matrix`).

---

## Архитектура пайплайна

1. **User Query** Поиск в базе знаний (SberQuAD).
2. **Hybrid Search:** `0.5 * BM25 (pymorphy3) + 0.5 * Dense (LaBSE)` Извлечение Top-3 релевантных чанков.
3. **Generator:** `Qwen2.5-7B-Instruct` формирует краткий и строгий ответ по контексту.
4. **Internal Validator (Critic):** Анализирует ответ и контекст, использует Chain-of-Thought (CoT) логику и выносит вердикт (`ДА` / `НЕТ`).
5. **Output:** Выдача верифицированного ответа либо заглушки: *"Ошибка: Ответ заблокирован модулем безопасности"*.
