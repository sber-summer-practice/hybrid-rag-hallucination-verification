# Данные проекта (Data)

В данном проекте используется русскоязычный датасет **SberQuAD**.

Физически скачивать датасет в эту папку не требуется, так как в коде пайплайна (`src/pipeline.py`) настроена его автоматическая загрузка напрямую из репозитория Hugging Face.

* **Ссылка на полный датасет:** [Hugging Face — SberQuAD](https://huggingface.co/datasets/kuznetsoffandrey/sberquad)
